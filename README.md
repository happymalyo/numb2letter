## Number2Letter

Number2Letter converts numbers (1â€“26) to their alphabetical equivalents in multiple languages, including Malagasy (MG), French (FR), and English (EN). It uses the [Tanisa](https://www.npmjs.com/package/tanisa) library for Malagasy translations and [to-words](https://www.npmjs.com/package/to-words) for French and English.

## Demo

Try the app here : [Numb2Letter](https://numb2letter.vercel.app/)

## Installation

Follow these steps to set up the project locally.

```bash
git clone git@github.com:happymalyo/numb2letter.git
cd numb2letter
npm install
npm run dev
```

Star the project if you like it. Enjoy ;)
